# StillLight MVP 范围

StillLight 的 MVP 目标是先完成一个真实可跑、可拍、可保存、可讲清楚的 iOS 胶片相机闭环。

## MVP 包含

- AVFoundation 实时相机预览
- 拍照
- 原生录像，支持音频、录制计时和保存到系统照片
- 前后摄像头切换
- 闪光灯：关闭 / 开启 / 自动
- 曝光补偿
- 点按对焦和测光
- 对焦动画
- 网格线、水平仪、比例取景框
- 比例：3:2、4:3、1:1、16:9、Half
- 当前胶卷和剩余张数
- 27 个胶片 / 经典机型预设
- 中文 / 英文 UI 切换
- 本地相册记录
- 拍照后冲洗预览
- 保存到系统照片
- JPEG 写入 StillLight 胶片元数据
- 可选保存原图
- 暗房多图导入
- 暗房冲洗当前 / 冲洗全部
- 暗房保存当前 / 保存全部
- 本地智能胶卷推荐
- 分享

## 主推胶卷

- Human Warm 400：人文、咖啡馆、室内、日常生活
- Shadow Walk 800：暗角氛围、街道、建筑、博物馆
- Soft Muse 400：柔和人像、肤色、日光和室内人像

## 图像处理 MVP

```text
图片输入
-> 方向修正和降采样
-> 中心裁切
-> 曝光
-> 色温 / 色调
-> 对比度 / 饱和度
-> Tone Curve
-> Halation
-> 暗角
-> 漏光
-> CPU 胶片颗粒
-> 日期戳
-> 胶片边框
-> 机型标注
-> JPEG 导出
```

## 暂不做

- 实时胶片预览
- 胶片效果录像导出
- RAW / HEIC 导出
- LUT 编辑器
- 云端多模态 AI
- 完整胶卷历史和云同步
- 商业级账号系统

## MVP 取舍

当前优先级是“真机可用”和“作品集可讲”。因此照片管线先使用 CoreImage + CPU 颗粒，录像先使用原生 `AVCaptureMovieFileOutput`。实时胶片预览和胶片视频会放到 Metal 路线里做，这样技术路线更稳，也更适合后续迭代。
