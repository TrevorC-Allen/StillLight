## 分类

- Featured：主推场景卷，服务人文、暗角氛围和人像
- Portrait：肤色和柔和反差优先
- Color Negative：日常彩负
- Classic Camera：经典相机 / 机型色彩和边框标注
- Instant：拍立得纸框和低反差
- Black & White：黑白纪实颗粒
- Digital：CCD / 早期数码卡片机
- Experimental：钨丝灯、玩具相机、强风格化

## 胶卷物件视觉

胶卷选择器不是普通滤镜列表，而是 Dazz-like 的物件抽屉。每个预设会映射为一种可识别的实物：

- 135 彩负纸盒
- 黑白胶卷暗盒
- 拍立得相纸包
- 120 / 实验胶片纸套
- 一次性相机
- 半格票据
- 经典机型 / CCD / 中画幅相机机身

每个预设在胶卷选择器中都有独立的 `FilmCoverStyle`，用于控制：

- 主色板
- 中央光晕颜色和位置
- 纸盒 / 相纸 / 暗盒材质
- 独立微型样片场景
- 确定性纸纹和磨损点
- 接触阴影和抽屉层次
- 镜头玻璃反光
- 短代码

微型样片场景会按胶卷 ID / 分类映射到咖啡桌、街影暗角、柔和人像、夜景霓虹、CCD 校园、拍立得桌面、海边、花草、半格日记等场景。这样不需要依赖大量外部样张素材，也能让用户在切换胶卷时直接看到“成片氛围”。

## 当前 Pipeline 重点

当前出片不是单纯套 LUT，而是由每个预设控制下列模块：

- 3200px 高质量处理路径
- 曝光、白平衡、色调、对比和饱和
- Tone Curve
- 高光 / 阴影调整
- FilmRenderingProfile：局部微对比、中间调柔化、高光恢复、肤色保护
- 胶卷专属色彩响应矩阵
- 高光遮罩 Halation
- 柔和径向镜头落光
- 稳定 seed 漏光
- 带肤色保护的颗粒和 finishing texture
- 纸框纹理、日期戳重影和机型标注

## 输出边框

`whiteFrame`、`paper` 和 `instant` 会在导出图上生成底部边框，并写入：

- 选中的机型 / 风格名称
- StillLight 胶卷短名
- ISO 标记
- 可选日期戳

这个设计让照片不是简单套滤镜，而是有“相机直出”和“胶卷冲洗”的仪式感。
