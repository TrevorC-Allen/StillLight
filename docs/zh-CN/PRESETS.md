# 胶卷预设系统

StillLight 当前内置 27 个原创胶片 / 机型风格。它们参考常见胶片、经典相机色彩、
早期数码相机和拍立得格式，但使用原创命名和本地渲染参数，不直接复刻商业品牌滤镜。

当前产品方向是 Dazz-like、场景优先、样张驱动：胶卷选择器应该像打开一只实体胶卷抽屉，
用户先通过物件和样片氛围理解风格，再看到参数。

## 分类

- Featured：主推场景卷，服务人文、暗角氛围和人像。
- Portrait：肤色安全、柔和反差和高光控制优先。
- Color Negative：日常彩负，适合旅行、家庭和街头。
- Classic Camera：经典相机 / 机型色彩和边框标注。
- Instant：拍立得纸框、低反差和偏暖纸感。
- Black & White：黑白纪实颗粒。
- Digital：CCD / 早期数码卡片机。
- Experimental：钨丝灯、玩具相机、强风格化。

## 胶卷物件视觉

胶卷选择器不是普通滤镜列表，而是 Dazz-like 的物件抽屉。每个预设会映射为一种可识别的
实物：

- 135 彩负纸盒
- 黑白胶卷暗盒
- 拍立得相纸包
- 120 / 实验胶片纸套
- 一次性相机
- 半格票据
- 经典机型 / CCD / 中画幅相机机身

每个预设在胶卷选择器中都有独立的 `FilmCoverStyle` 和 `FilmCoverComposition`，用于控制：

- 主色板和短代码
- 中央光晕颜色和位置
- 纸盒 / 相纸 / 暗盒 / 纸套 / 机身材质
- 3x3 摄影感 contact sheet 样张裁切
- 独立纸盒封面构图
- 确定性纸纹、包装噪点和磨损边缘
- 接触阴影和抽屉层次
- 相机物件的镜头玻璃反光

微型样片会按胶卷 ID / 分类映射到咖啡桌、暗角空间、窗边肖像、市集街拍、中画幅静物、
钨丝夜街、CCD 闪光餐桌、即影桌面和黑白夜景等 9 类摄影场景。胶卷抽屉使用中心吸附
carousel，滑动停止后焦点胶卷会自动居中，并提供轻触感反馈。

## 相机模型展示层

每个胶卷现在都有独立的 `FilmCameraProfile`，用于把“选择一卷胶卷”表达成“拿起一台相机”：

- `FilmPreset.cameraProfile`：单个胶卷的相机模型入口。
- `FilmLibrary.cameraProfiles`：按当前胶卷顺序返回 27 个相机 profile。
- `FilmLibrary.missingCameraProfileIds`：QA 检查，当前应为空集合。
- `CameraModelCategory`：袖珍机、旁轴、单反、中画幅、即影、一次性相机、玩具相机、电影机、CCD 卡片机、半格。
- `CameraBodyStyle`：控制机身材质 / 形态的稳定枚举，不依赖图片资产。
- `CameraAccessoryCapability`：默认配件能力标签，包括双曝、长曝、星芒、闪光、色温、录像、日期戳、自拍、微距、方构图、半格、宽幅。

## 渲染参数

每个胶卷同时定义产品信息和 pipeline 控制项：

- ISO、显示名、本地化名称、短名、机型 / 风格标注。
- 场景标签和本地化说明，用于选择器和推荐逻辑。
- 曝光偏移、色温偏移、Tint 偏移、对比度、亮度和饱和度。
- Tone Curve 和高光 / 阴影处理。
- 颗粒强度、颗粒尺寸、暗角、Halation 和漏光。
- 日期戳颜色和边框类型。
- 导出 JPEG 时写入的 StillLight 胶片元数据。

## 当前 Pipeline 重点

当前出片不是单纯套 LUT，而是由每个预设控制下列模块：

- 3200px 高质量处理路径。
- 曝光、白平衡、色调、对比和饱和。
- Tone Curve 和高光 / 阴影调整。
- `Film Rendering Profile`：局部微对比、中间调柔化、高光恢复和肤色保护 finishing。
- `Tone Separation`：阴影、中间调和高光拥有不同的色彩响应、反差响应和色彩矩阵倾向。
- 高光遮罩暖色 Halation。
- 柔和径向镜头落光。
- 稳定 seed 漏光。
- 亮度相关颗粒和 finishing texture。
- 纸框纹理、日期戳重影和机型标注。

## 输出边框

`whiteFrame`、`paper` 和 `instant` 会在导出图上生成底部边框，并写入：

- 选中的机型 / 风格名称
- StillLight 胶卷短名
- ISO 标记
- 可选日期戳

这个设计让结果更像一张从 StillLight 胶卷冲洗出来的照片，而不是匿名导出的滤镜图。

## QA 验收

MVP 预设验收：

- 27 个预设都出现在选择器中，并有正确分类和中英文文案。
- 三个主推胶卷在人文 / 暗角 / 人像方向中具有优先展示和真实样张验证。
- 切换胶卷会改变图像渲染、边框 / 机型标注和导出元数据。
- 胶卷物件在抽屉尺寸下能明显区分，且不依赖外部样张下载。
- 白框、相纸和拍立得边框中的机型 / 胶卷文字清晰可读。
- 相机拍摄和 Import Lab 对同一张图、同一胶卷、同一比例和强度输出一致。

下一步预设验收：

- 为每个主推胶卷补充 8-12 张真实 iPhone 样张，覆盖上面的样张方向。
- 用同一组样张对比当前程序化渲染和未来 3D LUT / `CIColorCube` 版本。
- 为肤色、高光恢复、阴影细节、暗角强度和颗粒可见度写出 pass/fail 记录。
- 新胶卷进入主库前，必须完成物件视觉、本地化文案、样张方向和 pipeline 参数 review。
